Loading Geocaching.com LOC files into a Garmin Vista Cx using gpsbabel

Crazy title, I know.

If you’ve got a Garmin Vista Cx, Legend Cx, Vista Cx or similar, and a Linux box, here’s how to load a set of geocaching.com cache locations onto the unit:

First, download and install gpsbabel from SourceForge. Make sure that you have the package libusb-devel installed so that you get the usb drivers correct.

Attach the Garmin via its USB cable.

rmmod garmin_usb
./gpsbabel -i geo -f geocaching.loc -o garmin -F usb:

That should do it!

Presumably this should work equally well on an OSX or Windows box as well, since gpsbabel is cross platform.
