Zuma clone for Ubuntu Linux

Just found out that there’s a great clone of the game Zuma for Ubuntu.

Enable the Universe repository and apt-get install zaz or use the Update Manager to install.  It’s got a pretty interesting dynamic and twist on the original Zuma.
